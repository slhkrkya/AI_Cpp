# AI C++ Terminal Assistant

## Proje Açıklaması
Bu proje, terminal üzerinden etkileşimli bir yapay zeka asistanı sağlar. Kullanıcılar, çeşitli komutlar aracılığıyla asistan ile etkileşimde bulunabilirler. Proje, komutlar aracılığıyla bilgi almayı, dosyalar üzerinde işlem yapmayı ve genel bir çalışma akışı yönetimini içerir.

## Kurulum
1. Projeyi klonlayın veya indirin.
2. Gerekli bağımlılıkları yüklemek için CMake kullanın.
3. Projeyi derlemek için terminal üzerinden aşağıdaki komutu çalıştırın:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

## Kullanım
Projeyi derledikten sonra, uygulamayı terminalden çalıştırabilirsiniz:
```bash
./ai_cpp_terminal
```
Uygulama başladığında `/help` komutu ile mevcut komutları görebilirsiniz.
